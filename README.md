## Hosting (GitHub Pages)

Enable **Settings → Pages → Deploy from branch → `main` / root**. The documents
will then be available at:

- `https://rahonok.github.io/terms-app/`
- `https://rahonok.github.io/terms-app/privacy-policy.html`
- `https://rahonok.github.io/terms-app/terms.html`

Use these URLs in App Store Connect and the Google Play Console.
